# 🗺️ Date Plan App

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-sharp)](http://makeapullrequest.com)

**Date Plan App** adalah aplikasi web interaktif yang dirancang khusus untuk membantu pasangan merencanakan kencan ideal mereka tanpa pusing. Aplikasi ini menyediakan kurasi ide aktivitas, pengaturan jadwal, estimasi anggaran (budgeting), hingga rekomendasi tempat berdasarkan preferensi pengguna.

## 🚀 Fitur Utama

- **Date Idea Generator:** Rekomendasi aktivitas kencan unik berdasarkan mood, cuaca, atau kategori (romantis, petualangan, hemat, dll).
- **Budget Planner:** Fitur kalkulator anggaran terintegrasi untuk melacak estimasi pengeluaran agar rencana tetap sesuai dompet.
- **Interactive Itinerary Builder:** Susun jadwal perjalanan kencan (*timeline*) dari jam ke jam dengan mudah.
- **Location & Place Recommendation:** Integrasi peta/rekomendasi kuliner dan tempat estetik terdekat.
- **Saved History & Wishlist:** Simpan rencana kencan masa depan (*bucket list*) atau arsipkan kenangan kencan yang sudah terlaksana.

## 🛠️ Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan ekosistem teknologi berikut:

- **Frontend:** React.js / HTML5, CSS3, JavaScript (Tailwind CSS)
- **Backend:** Node.js + Express.js / Python (FastAPI/Flask)
- **Database:** MongoDB / PostgreSQL / SQLite
- **API Integrations:** Google Maps API (untuk lokasi) / OpenWeather API (opsional)
